# TechSANS Visualizations

This repository contains the Jupyter notebook and supporting files used to generate visualization prototypes for the manuscript:

**_Early Prototyping of Visualization for Interpretable Passive Smartphone Digital Biomarkers with Cognitive and Behavioral Data_**  
(Forthcoming publication)

## Objective of the manuscript 
To provide a visualization approach and dashboard prototype for exploratory analysis of individual- and cohort-level smartphone sensor, and cognitive data across heterogeneous data streams.

## Getting Started
1. **Install required packages:**

```bash
pip install -r requirements.txt
```
2. **Launch Jupyter Lab or Notebook and open:**

TechSANS_viz_paper.ipynb

3. **Change the dataset file path in the code and run all cells to reproduce the visualizations used in the manuscript**
